# HÁBITAT PRIME S.A.S. · Entrega No. 2

## Contenido de esta entrega

Esta entrega incorpora la sección de **Inmuebles** con una vitrina profesional y buscador funcional.

Incluye:

- Catálogo inicial de inmuebles modelo.
- Filtros por operación, ciudad, tipo, presupuesto y habitaciones.
- Contador dinámico de resultados.
- Tarjetas premium de inmuebles.
- Ventana de detalle por inmueble.
- Botones de WhatsApp por propiedad.
- Estado vacío cuando no hay resultados.
- Estructura preparada para reemplazar inmuebles modelo por inmuebles reales.

## Archivos principales

- `index.html`
- `assets/css/styles.css`
- `assets/js/main.js`
- `assets/img/`
- `CNAME`

## Cómo actualizar en GitHub

1. Descomprime el archivo ZIP.
2. Entra al repositorio `habitat-prime-web`.
3. Sube todos los archivos de esta carpeta.
4. Reemplaza los archivos anteriores cuando GitHub lo pregunte.
5. Confirma con `Commit changes`.
6. Espera entre 1 y 3 minutos para que GitHub Pages publique la nueva versión.

## Cómo editar los inmuebles

Los inmuebles están en el archivo:

`assets/js/main.js`

Dentro del arreglo:

`const properties = [...]`

Ahí puedes cambiar:

- título;
- operación;
- tipo;
- ciudad;
- sector;
- precio;
- área;
- habitaciones;
- baños;
- parqueaderos;
- descripción;
- beneficios.
