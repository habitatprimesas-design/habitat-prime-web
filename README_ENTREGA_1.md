## Cómo actualizar en GitHub

1. Descomprimir este ZIP.
2. Entrar al repositorio `habitat-prime-web`.
3. Eliminar o reemplazar los archivos anteriores.
4. Subir todos los archivos de esta carpeta, incluyendo `index.html`, `assets` y `CNAME`.
5. Confirmar con **Commit changes**.
6. Esperar 1 a 3 minutos para que GitHub Pages publique.
