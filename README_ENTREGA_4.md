# HÁBITAT PRIME S.A.S. — Entrega No. 4
## Portal Propietarios y Arrendatarios

Esta entrega incorpora la primera estructura visual y funcional del futuro portal privado de clientes de HÁBITAT PRIME.

## Incluye

- Nueva página `portal.html`.
- Acceso visual para clientes.
- Panel demostrativo para propietarios.
- Panel demostrativo para arrendatarios.
- Módulos preparados para:
  - Centro documental.
  - Solicitudes de mantenimiento.
  - Pagos y reportes.
  - PQRS.
  - Notificaciones.
  - Soporte humano vía WhatsApp.
- Formulario de radicación rápida.
- SEO específico para portal.
- Navegación desde la página principal hacia el portal.

## Importante

El portal es una maqueta funcional de interfaz. Todavía no tiene autenticación real ni base de datos. Está preparado para integrarse después con:

- Supabase
- Firebase
- Zoho CRM
- HubSpot
- Google Sheets
- Airtable
- Backend propio
